# 创：时间 | TRON: TIME

> **一名程序员写给《创》系列的情书**

*献给凯文·弗林——他明白，最好的系统是用真心搭建的。*

---

## 这是什么

一部非商业性质的粉丝同人剧本，续写《创：战纪》（2010）之后萨姆·弗林与葵拉的故事。

由一名软件工程师创作，他在《创》系列对数字身份、系统架构与代码之魂的探索中，找到了深刻的共鸣。

本剧本试图填补《战纪》与《战神》之间的叙事断裂，致敬官方续集遗弃的角色与主题。

---

## 故事梗概

逃离电子网格后，萨姆与葵拉建立了新生活：地上是山地游乐园，地下是秘密数字实验室。但葵拉的 ISO 生物学特征在现实世界中开始崩解——熵增、引力、时间本身，都成了她的敌人。

他们的避难所被 MCP 的残余入侵，这个来自 1982 年的古老暴君，现已进化并充满复仇之意。在一次"充电周期"中，萨姆与葵拉被困在自己的网格里，必须打一场没有退路的数字战争。

在葵拉最黑暗的时刻，萨姆继承了父亲的力量——重塑系统的能力。但胜利揭示了一个更深层的真相：克鲁幸存了下来，被弗林最后的宽恕之举释放。而他正在寻找他们。

剧本通过后彩蛋场景与《创：战神》相连，将三代程序与用户编织成单一的传承。

## 为何存在

迪士尼的《创：战神》（2025）边缘化了萨姆与葵拉，用新主角取代了他们的故事。这部剧本是一个粉丝的回应——一封写给以下主题的情书：

- **创造与后果**（弗林的双重遗产）
- **数字灵魂与生物无常**（葵拉的 ISO 本质）
- **超越二进制的救赎**（克鲁的转变）
- **时间作为终极系统**（不朽程序与衰老用户的悲剧）

---

## 技术笔记

作为一名程序员，我以设计系统时的同样逻辑来构建这部剧本：

- **架构**：三幕结构，镜像网格的层级（表面/避难所/核心）
- **状态管理**：角色弧作为状态机（萨姆：否认→接受→继承）
- **错误处理**：MCP 作为遗留技术债务，重新浮现以摧毁新系统
- **文档**：每个场景都服务于核心协议——时间，以及我们对抗它而建造的一切
